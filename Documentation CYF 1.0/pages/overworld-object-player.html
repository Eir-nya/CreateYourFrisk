The Overworld: The Player Object


This Player class is not the same as the one used in battle! This one is specific to the overworld!



Player.GetLevel() Returns the player's level (LVL).


Player.SetLevel(int value) Sets the player's level (LVL).


Player.GetHP() Returns the player's HP.


Player.SetHP(number value) Sets the player's HP.


Player.GetMaxHP() Returns the player's max HP.


Player.SetMaxHP(int value) Sets the player's max HP.


Player.GetName() Returns the player's name.


Player.SetName(string value) Sets the player's name.


Player.GetWeaponATK() Returns the player's weapon's ATK.


Player.GetArmorDEF() Returns the player's armor's DEF.


Player.GetATK() Returns the player's total ATK.


Player.GetDEF() Returns the player's total DEF.


Player.GetGold() Returns the player's amount of gold.


Player.SetGold(int value) Sets the player's amount of gold.


Player.GetWeapon() Returns the player's weapon's name.


Player.SetWeapon(string value) Makes the player equip the weapon given in value, but only if a weapon with that name is in their inventory.


Player.GetArmor() Returns the player's armor's name.


Player.SetArmor(string value) Makes the player equip the armor given in value, but only if any armor with that name is in their inventory.


Player.GetEXP() Returns the player's EXP.


Player.SetEXP(int value) Sets the player's EXP to value.


Player.ForceHP(number value) Sets the player's hp. But this time, the player's HP can go up to 1.5 times their max HP.


Player.Hurt(int value) Removes value HP from the player's current HP. The Player can't be killed by use of this function.


Player.Heal(int value) Adds value HP to the player's current HP, but can not exceed the player's maximum HP.


Player.CanMove(bool canMove) This function determines whether the player can move or not.
However, if canMove is set to true and the function is called while a main event is running, then you can't trigger any other main events, teleport to other maps or open the menu until the main event is stopped.


Player.Teleport(string mapName, float posX, float posY, int direction = 0, bool noFadeIn = false, bool noFadeOut = false) Teleports the player to the map named mapName.
You can set the player's direction using direction.
noFadeIn and noFadeOut determine whether there will be fades before or after the teleport, respectively.
